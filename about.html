About this website

README is a repository for resources, tutorials and workshops by Kirsten Spruit, focussing on designing and (self-)publishing research — whether in printed or web-based form.


The README file

The title and design of this repository play with the idea of the README, a text file — usually in simple plain text — used in software development, that contains information about the other files in a directory or archive of computer software.

With its filename written in uppercase to stand out and have prominence, a README text is to be read before starting an installation or process of sorts. It typically serves as a form of documentation that introduces and explains a project, often providing instructions to be followed along. Its general purpose is to offer guidance for others to engage with whatever it refers to.


Expanding the README

Here the README is explored as a means to start and guide the process of creating a publication, from a prompt for ideation to a personal handbook to collect design choices, notes-to-self and helpful references.
